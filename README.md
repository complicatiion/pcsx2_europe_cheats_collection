# pcsx2\_europe\_cheats\_collection



This e.pnach cheat collection mainly refers to EU versions of PC2 games, i.e. not NTSC SLUS games but often PAL SLES games. There are special .pnach files dedicated to country-specific games from France, Germany and Spain, as well as some Japanese ports.



# Notes:



The .pnach files are first marked with their region code (often SLES) and then with the corresponding game-specific CRC.

This means you can use the game properties in spcsx2 to check exactly which games you have and which are compatible. If the game is compatible with .pnahc cheta, it will appear in the game properties in pcxs under ‘cheats’.



Example: SLES-53558\_1FA82CDF.pnach = Need for Speed - Most Wanted (PAL)



Many of the cheats are compatible with other versions/regions, so you can try renaming them to your region code and CRC, and with luck they will work.
